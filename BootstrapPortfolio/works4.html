Wall Bounce and Color Change

For this assignment I was tasked to use a box in Java with set boundaries to enable a ball to bounce off the walls and change color. This was from CSCI 121 Intro to Programming Java II.

Bounce and Color Change

Java

With this task, I created a bounded box that let a ball bounce off the walls and change color when it bounced. In this example, after the first bounce it turned green. This aided me to be more knowledgable of the boundaries of objects, like this box and ball, in Java, and how to use it to my advantage. It also helped me gain more knowledge of the various ways you can make objects dynamically change in Java.
